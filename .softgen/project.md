# Admin Dashboard

## Vision
A vibrant, high-contrast admin dashboard for managing products, posts, orders, categories, and shop information. Built for frontend demonstration with localStorage persistence.

## Design
**Color System:**
- `--background: 222 47% 11%` (deep slate)
- `--foreground: 210 40% 98%` (crisp white)
- `--primary: 243 75% 59%` (vibrant indigo)
- `--accent: 348 83% 61%` (bright coral)
- `--muted: 217 33% 17%` (slate muted)
- `--card: 222 47% 11%` (slate card)
- `--border: 217 33% 17%` (subtle borders)

**Typography:**
- Heading: Inter (600-700 weight)
- Body: Inter (400-500 weight)

**Style Direction:** Electric command center - modern SaaS with cyberpunk energy. High contrast, vibrant accents, card-based layouts, generous whitespace.

## Features
- Dashboard overview with key metrics
- Product management (CRUD with categories)
- Post/content management
- Order tracking and management
- Category organization
- Shop settings and information
- Sidebar navigation with icons
- Data tables with search/filter
- Forms with validation
- Mock data persistence via localStorage
